# Gerador de Senhas 🔒

### ✏️ Descrição
O projeto é um gerador de senhas, onde o usuário escolhe a quantidade de caracteres e o programa gera uma senha aleatoria
<br>
Exercício para aperfeiçoar minhas habilidades em Javascript
